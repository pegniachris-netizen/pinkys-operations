## Operations System Components

### 1. Content Calendar (Notion Database)
- **30-day rolling content schedule** across 9 channels
- **Market rotation**: Monday=Buffalo, Tuesday=Tampa, Wednesday=Sarasota (repeating)
- **9 Channels**: TikTok Business, Facebook, Instagram, LinkedIn Company, Threads, Chris Pegnia LinkedIn Personal, Chris Pegnia YouTube, chrispegnia TikTok Personal, Learning with Leaders Instagram
- **Automated via**: Buffer scheduling + Zapier triggers

### 2. Lead Tracker (Notion Database)
- Full CRM tracking: Name, Company, Market, Service Needed, Source, Status, Phone, Email, Value Estimate, Next Action, Date
- **Sources**: Website, Referral, Social Media, ConstructConnect, SmartBid, Procore, Direct Call, Email, Networking Events
- **Pipeline stages**: New → Contacted → Qualified → Proposal Sent → Negotiating → Won/Lost

### 3. Bid Pipeline (Notion Database)
- Construction bid tracking across all platforms
- **Platforms**: ConstructConnect, SmartBid, Procore, Direct
- **Stages**: Identified → Reviewing → Estimating → Submitted → Won/Lost/No Bid
- Deadline and submission date tracking with value estimates
