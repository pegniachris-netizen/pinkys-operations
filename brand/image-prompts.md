# AI Image Generation Prompts

> Template library for generating brand-compliant visual content for Pinky's Property Management.

---

## Universal Prompt Elements

Every image generation prompt must include these elements:

```
Required: "hot pink color #FF69B4"
Required: "pink flamingo logo watermark bottom right corner"
Required: "professional photography, high resolution, vibrant colors"
Preferred: "sunny day, clean composition, rule of thirds"
Avoid: "text overlays, other watermarks, competitor logos, dark/moody"
```

## Prompt Modifiers

Add these to any base prompt for specific effects:

| Effect | Modifier Text |
|--------|--------------|
| Aerial/drone | "aerial drone photography, bird's eye view, DJI quality" |
| Golden hour | "golden hour lighting, warm tones, long shadows" |
| Action/motion | "motion blur background, sharp subject, dynamic angle" |
| Dramatic | "dramatic lighting, high contrast, cinematic composition" |
| Clean/minimal | "clean white space, minimal composition, product photography" |
| Documentary | "documentary style, candid moment, authentic feel" |
